# devtools
Collection of tools that I used to be more productive as Software Engineer

## IDE & Text Editor

- [LunarVim](https://www.lunarvim.org/) - text editor that I used for coding in general
- [VSCode](https://code.visualstudio.com/) - IDE that I used if LunarVim is not suitable
- [SublimeText](https://www.sublimetext.com/)- text editor that I used for some adhoc task

## Terminal

### Shell

- [zsh](https://www.zsh.org/) - UNIX shell, my default shell
- [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh) - to manager my zsh config
- [ranger](https://github.com/ranger/ranger) - terminal file manager with VIM experience

### Git

- [git](https://github.com/git/git) - version control

### HTTP

- [curl](https://github.com/curl/curl) - CLI for transferring data using various network protocols

### Documentation

- [tldr](https://tldr.sh/) - command cheatsheet

### Others

- [tmux](https://github.com/tmux/tmux) - terminal multiplexer
- [chezmoi](chezmoi.io/) - to manager dotfiles
- [docker](https://www.docker.com/) - containerization tool
- [kubectl](https://kubernetes.io/docs/reference/kubectl/) - kubernetes CLI tool
- [skaffold](https://skaffold.dev/) - CLI for continous development for container based & kubernetes applications

## GUI Tools

- [Postman](https://www.postman.com/) - to test API, GraphQL, etc..
- [dbeaver](https://github.com/dbeaver/dbeaver) - Database client
- [Docker Desktop](https://www.docker.com/products/docker-desktop/) - to easily manage docker
